# 🌟 TipidLitro
A mobile fuel-management and trip-planning app that helps Filipino public transport drivers and delivery riders spend less, earn more, and travel smarter.

## 🚀 About the Project
This project is developed as part of **SparkFest 2026**, a hackathon organized by **Google Developer Groups on Campus – Polytechnic University of the Philippines (GDG PUP)**.
It aims to solve a real-world or community-based problem through technology, innovation, and collaboration.

## 🎯 Problem Statement
The Philippines' heavy reliance on imported petroleum leaves it exposed to global oil price shocks, driving up transportation, logistics, and delivery costs nationwide. Public transport drivers and delivery riders are hit hardest — often working longer hours as fuel costs outpace fare adjustments. With fuel prices shifting week to week, over 110 million Filipinos face rising inflation and shrinking margins, with little time to adapt. Efficient fuel use and smarter trip planning have never been more critical.

## 💡 Proposed Solution
TipidLitro turns volatile fuel prices into clear, personalized guidance for everyday drivers.
- Calculates real trip costs, distances, and travel times based on actual driving routes
- Estimates driving range from current fuel level, so drivers know how far they can go before refueling
- Locates the nearest gas stations using live map data, sorted by distance
- Includes an AI assistant (Litroy) that answers fuel and trip questions in natural language — no need to navigate menus or calculators manually
- Built mobile-first and optimized for low-end smartphones, the reality for most drivers and riders in the target community

## ⚙️ Features
- Trip Cost & Time Calculator — real driving-route fuel cost, distance, time, and liters needed
- Fuel Range Calculator — estimated driving range from current fuel level, shown on a dashboard-style gauge
- Gas Station Locator — nearby stations sorted by distance, with one-tap directions
- AI Assistant — conversational trip/fuel help in English, Tagalog, or Taglish, powered by Gemini

## 🧪 Tech Stack
- **Frontend:** HTML, CSS, JavaScript, Leaflet.js
- **AI/APIs:** Google Gemini API, OpenStreetMap Nominatim, OSRM, Overpass API, Browser Geolocation API
- **Tools:** Google Fonts, Vercel, GitHub

## 🔵 Google Technologies Used
- **Gemini** — AI trip planning, budget advice, and personalized fuel-saving tips, turning raw data into decisions drivers can act on
- **Google Maps API** — routing, fuel-efficient directions, station search, and traffic data to cut wasted fuel and travel time
- **Firebase** — account logins, chat history, and real-time community fuel reports, syncing live station conditions across users
- **Google Authentication** — simple, secure sign-in using Google accounts
